# Retail Sales Performance Analysis (SQL)

## Project Overview
This project analyzes a retail transaction dataset containing approximately 9,800 orders to identify key drivers of revenue. SQL queries were used to explore sales performance across regions, product categories, customer segments, and time periods.

## Dataset
The dataset contains retail sales transaction data including:

- Order information
- Customer details
- Product categories and subcategories
- Geographic information (region and state)
- Sales values for each transaction

## Analysis Performed
Using SQL queries, the following analyses were conducted:

- Total number of orders and total revenue
- Sales performance by region
- Sales by product category and subcategory
- Top customers and top products by revenue
- Sales by customer segment
- Sales by shipping method
- Top states by total sales
- Sales trends by year

## Key Insights
- Certain regions generate significantly higher sales than others.
- A small number of customers account for a large share of total revenue.
- Specific product categories and subcategories dominate overall sales performance.
- Sales activity is distributed across multiple geographic markets and customer segments.
